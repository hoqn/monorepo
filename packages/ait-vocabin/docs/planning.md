# 앱인토스 독일어 단어 학습 앱 기획서

> 앱 가칭: **VocaBin** (미정)
> 작성일: 2026-03-21
> 상태: 초안

---

## 1. 서비스 개요

### 핵심 가치 제안
독일어 단어 학습을 **훌륭한 인터랙션과 UX**로 차별화한 앱. 학습 자체가 즐겁고 감각적인 경험이 되도록 설계한다.

### 한 줄 설명
스페이스드 리피티션 기반의 독일어 명사 학습 앱. 뛰어난 인터랙션과 게임적 요소로 꾸준한 학습을 유도한다.

### 차별화 포인트
- 캐릭터 마스코트 없이 **인터랙션 자체로 감성**을 살림 (햅틱 피드백, 카드 스와이프 등)
- **Spaced Repetition** 알고리즘을 전면 어필 (망각 곡선 기반 복습)
- 목숨을 잃어도 강제 종료 없이 **세션 내 자연스러운 복습으로 회복** 가능

## 2. 타겟 사용자

- **주**: 독일어 완전 초보 (알파벳은 알지만 단어는 모르는 수준)
- **부**: 중급자가 틈틈이 복습용으로 활용

---

## 3. MVP 범위

### MVP (v1)
- 독일어 명사: **성(der/die/das) 맞추기** + **복수형 맞추기** 퀴즈

### 이후 확장
- 동사 형변화 (현재형, 과거형 등)
- 다른 언어 지원 (프랑스어, 스페인어 등)
- 문장 단위 학습

---

## 4. 학습 설계

### 4.1 데이터셋
- **소스**: Wiktionary (추후 다국어 확장을 고려한 선택)
- 파싱 대상: 독일어 명사의 성, 복수형 데이터

### 4.2 퀴즈 형식

**명사 성 맞추기**
- 단어 제시 → `der` / `die` / `das` 세 가지 큼지막한 카드 버튼 선택

**명사 복수형 맞추기**
- 단어 제시 → 복수형 객관식 선택 (선택지 수 TBD)

### 4.3 세션 구조
- 듀오링고 수준의 세션 길이 (~10–15문항, 약 5–10분)
- **하루 제한 없음** (원하는 만큼 세션 가능)
- 세션 내 순서: 신규 단어 학습 → 복습 문항 혼합

### 4.4 복습 알고리즘
- **Spaced Repetition** 적용 (예: SM-2 알고리즘 기반)
- 정답/오답에 따라 다음 복습 주기 자동 조정
- 서비스 어필 포인트로 활용

## 5. 목숨 시스템

| 항목 | 내용 |
|------|------|
| 초기 목숨 수 | 5개 |
| 오답 시 | 목숨 1개 감소 |
| 목숨 소진 시 | 세션 강제 종료 없음 → **내부 복습 서브세션** 진입 |
| 복습 서브세션 | 틀린 단어 위주로 재학습, 완료 시 목숨 회복 후 원래 세션 복귀 |
| 목숨 회복 (서브세션 외) | 추후 검토 (광고 시청 등) |

---

## 6. 게임화 요소

### 스트릭 (Streak)
- 매일 1개 이상 세션 완료 시 스트릭 유지
- 스트릭 끊김 방지 아이템 등 추후 고려

### 경험치 & 레벨 (XP / Level)
- 세션 완료, 정답, 스트릭 달성 등으로 XP 획득
- XP 누적에 따라 레벨 상승
- 세부 수치는 밸런싱 단계에서 결정

### 리더보드
- 범위: 전체 유저 (주간 리셋)
- 기준: 주간 XP 합산

### 배지 (Badge)
- 시스템만 구현 (MVP)
- 조건 디테일은 추후 결정
- 예시 후보: 첫 세션 완료, 7일 스트릭, 특정 레벨 달성, N개 단어 학습 등

## 7. UX / 인터랙션 방향

- **햅틱 피드백**: 정답/오답 시 다른 진동 패턴
- **카드 스와이프**: 단어 카드를 스와이프하는 제스처 인터랙션 (세부 방식은 디자이너와 협의)
- **화면 전환 애니메이션**: 자연스럽고 감각적인 전환
- 캐릭터 마스코트 없이 인터랙션 자체로 감성 표현
- 세부 인터랙션 설계는 기획 확정 후 디자이너와 협업
